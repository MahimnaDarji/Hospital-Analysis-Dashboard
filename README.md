# Hospital Analysis Dashboard – Power BI Project

## Project Overview

This project is a multi-page Hospital Analysis Dashboard built using Power BI. It provides a structured view of hospital operations, covering patients, doctors, hospital management, and finance.

The dashboard is designed to help management monitor performance, track revenue, analyze patient flow, and understand resource utilization in one unified reporting system.

## Project Highlights

- Multi-page dashboard with dedicated modules: Overview, Patient, Doctor, Hospital, and Finance  
- Combined operational and financial reporting  
- Clear KPI-driven layout for decision-making  
- Revenue breakdown by charge type  
- Bed occupancy and surgery tracking  
- Medicine and supplier stock monitoring  
- Doctor-level performance and commission analysis

## 2. Overview Page

### Purpose

The Overview page provides a high-level snapshot of hospital performance across operations and finance.

### Questions It Answers

- How many patients, doctors, and staff are active?
- What is the total billing volume?
- What is the bed occupancy rate?
- How are patient discharges trending?
- What is the stock versus sale balance?
